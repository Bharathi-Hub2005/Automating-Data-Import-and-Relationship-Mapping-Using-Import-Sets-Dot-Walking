## Project Title
### Automating Data Import and Relationship Mapping Using Import Sets & Dot Walking

---

## Project Overview

This project demonstrates how to efficiently import external employee data into ServiceNow using **Import Sets** and **Transform Maps**, while maintaining proper data relationships (especially reference fields like Department and Manager). We also implemented **Dot Walking** on the Incident form to auto-populate related user details, improving usability and reducing manual effort.

## Technologies / Features Used

- **Google Sheets** – Data Source preparation
- **ServiceNow Import Sets** – Staging and loading external data
- **Transform Maps & Mapping Assist** – Field mapping between tables
- **Reference Field Mapping** – Department, Manager
- **Dot Walking** – `Assigned to.department`, `Assigned to.email`, `Assigned to.manager`
- **sys_user table** integration with **Incident table**
